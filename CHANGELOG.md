## [0.5.0] - 2026-06-26

### 🚀 Added
- 📘 **Project Overview Page** — Comprehensive `overview.html` explaining project goals, the Natas wargame, and target audience
- 🔗 **Overview Link in Header** — Quick access to the overview page from both English and Persian main indices
- 🇮🇷 **Complete Persian Translation** — All 35 levels fully translated into Farsi with full RTL support
- 🎨 **Vazirmatn Font** — Added Persian font support for better typography and readability
- 📝 **Comprehensive Documentation** — Added `CHANGELOG.md`, `natas-structure.txt`, and `README.md`

### 🔄 Changed
- 🔗 **Fixed Navigation Links** — Corrected relative paths in overview page (changed from `../index.html` to `index.html`)
- 📱 **Improved Mobile Responsiveness** — Better layout for sidebar, grid, and typography on smaller screens
- 🖌️ **UI Refinements** — Enhanced visual consistency across English and Persian versions
- 🎯 **Content Expansion** — Enhanced explanations and added more practical examples

### 🐛 Fixed
- 🔗 **Link Paths** — Fixed broken navigation between overview and index pages
- 📐 **RTL Support** — Corrected right-to-left layout issues in Persian version
- 📊 **Sidebar Navigation** — Improved scroll tracking and level number display
- 📱 **Responsive Issues** — Fixed various mobile layout problems

## [0.4.0] - 2026-06-23

### 🚀 Added
- 🌍 **Bilingual Structure** — Separated English and Persian versions into `en-natas/` and `fa-natas/` directories
- 🔄 **Language Switcher** — Easy toggle between English and Persian versions in the header
- 📂 **Organized File Structure** — Clean separation of language-specific assets and levels

### 🔄 Changed
- 📁 **Reorganized Project** — Restructured folders for better maintainability
- 🔗 **Updated Navigation** — All internal links now support bilingual switching

## [0.2.0] - 2026-06-17
