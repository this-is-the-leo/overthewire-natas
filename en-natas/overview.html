What is Natas?

Natas is a web security wargame from OverTheWire that teaches the fundamentals of web application security through a series of progressive challenges.

// About the Game

Natas is a web security wargame designed to teach the fundamental concepts of web application security. Each level presents a vulnerable web application with a specific security flaw. The goal is to find and exploit the vulnerability to retrieve the password for the next level.

The challenges start with basic concepts like viewing HTML source code and gradually progress to more advanced topics including SQL injection, command injection, cross-site scripting (XSS), file inclusion, deserialization vulnerabilities, and XML External Entity (XXE) attacks.

Unlike many CTF platforms, Natas is designed to be accessible to beginners while still providing valuable learning experiences for experienced developers. Each level builds on previous knowledge, creating a structured learning path through the landscape of web security vulnerabilities.

// Why This Writeup Exists

This writeup was created to serve as a comprehensive learning resource for anyone going through the Natas wargame. While solving the challenges is rewarding, understanding the underlying vulnerabilities and learning how to prevent them is the true value of the exercise.

Each level in this writeup includes:

  • ▸ A detailed explanation of the vulnerability
  • ▸ Step-by-step solution instructions with multiple approaches
  • ▸ Code analysis of the vulnerable source code
  • ▸ Practical defense measures to prevent the vulnerability
  • ▸ Key takeaways to reinforce the learning

Whether you're a beginner learning web security for the first time or an experienced developer looking to deepen your understanding, this writeup aims to provide clear, actionable knowledge.

// What You'll Learn

By working through these levels and following along with this writeup, you will develop a practical understanding of the most common web security vulnerabilities. You'll learn to identify, exploit, and prevent issues including:

🟢 Fundamentals
Starting from the very basics, you'll learn how to view and analyze source code, bypass client-side restrictions, and discover hidden files and directories. These foundational skills are essential for any security work.
🟠 Server-Side Vulnerabilities
Dive deep into server-side security flaws including:
  • Local File Inclusion (LFI) — Reading arbitrary files through insecure file inclusion
  • Path Traversal — Escaping the intended directory structure
  • Command Injection — Executing system commands through unsanitized input
🔴 SQL Injection
Master one of the most critical web vulnerabilities:
  • Authentication Bypass — Breaking into applications
  • Blind SQL Injection — Extracting data without visible output
  • Time-Based Attacks — Using delays to infer information
  • UNION-Based Injection — Combining queries to extract data
  • Stacked Queries — Executing multiple statements
🔵 Advanced Exploitation
Explore sophisticated attack vectors:
  • PHP Type Juggling — Exploiting loose comparison quirks
  • Object Injection — Leveraging deserialization flaws
  • Log Poisoning — Turning server logs into code execution
  • XML External Entity (XXE) — Reading files through XML parsers
  • Session Attacks — Predicting and manipulating session IDs
🟣 File Upload Attacks
Understand the risks of file upload functionality:
  • Client-Side Validation Bypass — Circumventing frontend checks
  • Server-Side Validation Bypass — Evading image verification
  • Magic Bytes Exploitation — Using file headers to hide payloads
